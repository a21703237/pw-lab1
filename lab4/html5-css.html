CSS

CSS is a style sheet language used for describing the presentation of a document writen in a markup language such as HTML

Tipos de seletores CSS
universal check Used for the websites background color and font
tipo check h3 header will have a red color

id check used to change the font of one sentence in index
atributo check links that have the character "in" will have a blueish background
pseudo-classe check used to change the colors of links
pseudo-elemento check Used to make every first letter in a sentence to be bigger
